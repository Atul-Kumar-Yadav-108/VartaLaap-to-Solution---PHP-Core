# VartaLaap-to-Solution - PHP Core

**VartaLaap-to-Solution** is a Core PHP based Q&A website that allows users to ask doubts, provide solutions, and interact with the community, similar to **StackOverflow** or **Quora**.

## ðŸš€ Features

- **User Authentication:**
  - Sign Up / Login / Forgot Password (works before and after login)
  - Password visibility toggle (eye button)
- **Home Page:**
  - Display questions with **infinite scroll** (6 records load at a time)
- **User Interaction:**
  - Logged-in users can **ask questions**
  - Users can **comment** on questions
- **User Dashboard:**
  - View, edit, and delete own questions
  - Pagination for own question list
  - Search functionality
- **Profile Management:**
  - Users can **view and edit profile**
  - Upload **profile picture**
  - Logout functionality
- **SEO Optimized:**
  - Dynamic page titles and meta descriptions
- **Security Measures:**
  - Implemented basic security for user data and forms
